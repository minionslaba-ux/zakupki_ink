# Shtark Flow — мультикомпанийный финансовый портал (ERP/BI SPA)

Single Page Application на чистом JavaScript (ES6+), разворачивается на GitHub Pages,
взаимодействует с облачным бэкендом (Firebase Firestore) в реальном времени.

> Рабочая production-сборка — единый самодостаточный файл **`Калькулятор NEV.standalone.html`**
> (генерируется из исходника `Калькулятор NEV.dc.html`). Каталог `js/`, `css/`, `assets/`
> ниже — **референс-модули**: канонический, читаемый код ядра (контроллер пересчёта,
> аналитический движок, FIFO) для сопровождения и code-review. Логика идентична живой сборке.

## Поток данных (каскад пересчёта)

```
Курсы ЦБ/Биржа → Landed Cost (Закупки) → Себестоимость 1 кг (Производство, по-дням)
   → Маржа GM1/GM2 (Продажи, НДС + ₽/кг) → Баланс «Доставка» → KPI/Рейтинг ★
```

Гибридный учёт себестоимости переключается по `company_id`:
- **Компания №1 (партионный):** списание по фактическим партиям закупки (уникальная Landed Cost).
- **Компания №2 (FIFO):** списание по хронологии приходов 1С (`upload_timestamp ASC`),
  средневзвешенная себестоимость 1 кг на дату выпуска.
